# 说明

在开发1.0版本的时候，主要是考虑到自己项目的使用。因此很多朋友说他们有多个账户，配置文件该怎么写？

有的配置文件以前在redis中，或者db中，又该如何完成？

本次2.0版本解决了以上问题，并且提供了更加简单的调用接口。

1.0发出后，虽然配备了示例代码，还是有很多朋友无法灵活运用于代码中。

本次开发过程中将配置完善的文档，会介绍使用，以及此SDK开发思路，便于大家自己根据情况修改或者增加新功能。
